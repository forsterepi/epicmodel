# Rain example SCC model

An example SCC model created from `steplist_rain`.

## Usage

``` r
scc_rain
```

## Format

An object of class `epicmodel_scc`, which is a list of 10 elements. See
[`new_scc()`](https://forsterepi.github.io/epicmodel/reference/epicmodel_scc.md)
for the detailed structure of `epicmodel_scc` class objects.
