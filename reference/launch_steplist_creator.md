# Launch steplist creator `shiny` app

Run this function to start the Steplist Creator `shiny` app.

## Usage

``` r
launch_steplist_creator()
```

## Value

The `launch_steplist_creator` function is used for the side effect of
starting the Steplist Creator `shiny` app.

## Examples

``` r
if(interactive()){
launch_steplist_creator()
}
```
