# Birthday party example steplist

An example steplist, which contains the steps that tell Clara, under
which conditions her birthday party will be a success.

## Usage

``` r
steplist_party
```

## Format

An object of class `epicmodel_steplist`, which is a list of 8
data.frames. See
[`new_steplist()`](https://forsterepi.github.io/epicmodel/reference/epicmodel_steplist.md)
for the detailed structure of `epicmodel_steplist` class objects.
